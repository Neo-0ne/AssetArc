
Token Control Bot
------------------

Files:
- main.py: Core logic to manage and generate advisor tokens.
- controller.py: CLI interface to interact with the bot.
- advisor_tokens.json: Data file storing tokens.
- __init__.py: Marks this directory as a Python package.

Usage:
1. Run controller.py to start the interface.
2. Enter an advisor ID to generate a token.
3. Tokens are stored in advisor_tokens.json.

Make sure Python 3.x is installed.
