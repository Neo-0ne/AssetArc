FAQ / NLP Help Bot
=======================

This bot uses basic NLP to match user input to the closest FAQ entry and suggest the most appropriate bot.

Files:
- faq_nlp_help_bot.py — Core bot logic
- faqs.json — Sample FAQ knowledge base
- README.txt — This file

Instructions:
1. Extend `faqs.json` with real FAQs.
2. Run `faq_nlp_help_bot.py` and enter a natural language query.
3. The bot will return the best matched FAQ and suggest a bot to use.
